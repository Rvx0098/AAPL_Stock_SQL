# Apple Stock Analysis using PostgreSQL

## Project Overview

This project analyzes historical Apple (AAPL) stock market data using PostgreSQL.

The dataset contains historical trading records including stock prices and trading volume. SQL was used to perform exploratory data analysis, financial trend analysis, ranking, and time-series analytics.

## Tools Used

- PostgreSQL
- pgAdmin
- SQL
- Git & GitHub

## Dataset Columns

- Date
- Open
- High
- Low
- Close
- Adjusted Close
- Volume

## SQL Analysis Performed

- Total Trading Days
- Dataset Date Range
- Highest Closing Price
- Highest Volume Trading Days
- Largest Daily Gains
- Largest Daily Losses
- Average Closing Price by Year
- Top Performing Years
- Monthly Average Closing Price
- Stock Price Ranking using Window Functions
